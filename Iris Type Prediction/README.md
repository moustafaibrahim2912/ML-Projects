# Iris Flower Prediction

## Overview
This project builds a simple machine learning classifier to predict iris flower species based on basic flower measurements.

The goal is to demonstrate a clean and minimal end-to-end prediction workflow.

## Dataset
The project uses the built-in Iris dataset provided by Scikit-learn.

The dataset contains measurements of flower characteristics:

- sepal length
- sepal width
- petal length
- petal width
- species (target variable)

No external dataset files are required.

## Project Structure

notebooks:
1. Data loading, preprocessing, and model training in a single notebook

## Modeling Approach

A lightweight classification model is trained to distinguish between iris species.

## Model Performance

The model achieves high accuracy on this small dataset, demonstrating clear class separation.
