# Titanic Survival Prediction

## Overview
This project builds a machine learning classifier to predict passenger survival on the Titanic ship using demographic and travel data.

The goal is to demonstrate a compact end-to-end prediction workflow in a single notebook.

## Dataset
The project uses  Titanic passenger dataset.

The dataset includes features such as:

- passenger class
- age
- sex
- fare
- family relations onboard
- embarkation port
- survival (target variable)

Data cleaning and preprocessing were applied before modeling.

## Project Structure

notebooks:
1. Data loading, preprocessing, model comparison, and final training in a single notebook

## Modeling Approach

Multiple classification models were tested.  
Logistic Regression produced the best performance and was selected as the final model.

Additional models may be explored in future updates.

## Model Performance

The final model achieves strong accuracy and generalization on the dataset.

## Conclusion

This project demonstrates a concise machine learning pipeline, including preprocessing, model comparison, and final selection in a single workflow.
